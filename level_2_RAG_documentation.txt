These imports are your major upgrades.

### BM25 Retriever

```python
from langchain_community.retrievers import BM25Retriever
```

Purpose:

**Keyword-based retrieval**

Think:

### Traditional search engine

If user asks:

> N11 interface

```text

```

even if semantic embedding misses it.

Very useful in telecom.

Because 3GPP contains:

```text

N2
N11

SMContext
```

Exact terminology matters.

---

### Cross Encoder

```python
from sentence_transformers import CrossEncoder
```

This is your smartest improvement.

Purpose:

### Reranking retrieved documents

Instead of trusting retrieval blindly:

It re-checks:

> Which chunk truly answers the question?

---

Example

Question:

```text
Explain AMF registration
```

Retrieved chunks:

Chunk A:

```text
AMF registration flow
```

Chunk B:

```text
AMF scaling architecture
```

Chunk C:

```text
UE registration procedure
```

Vector DB may rank imperfectly.

CrossEncoder rescoring:

```text
Chunk C → 0.98
Chunk A → 0.90
Chunk B → 0.32
```

Then sorts by score.

This significantly improves answer quality.

---

### Conversation Memory

```python
from collections import deque
```

Used for:

### Chat continuity

Without memory:

User:

> Explain AMF

Next question:

> What about authentication?

LLM forgets AMF context.

With memory:

It understands:

> authentication in AMF flow
